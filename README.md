# Лабораторна робота №7-8 з Python

**Виконав:** Качоровський Данііл (КІб-1-24-4.0д)

## Опис

Цей репозиторій містить виконання завдань лабораторної роботи №7-8 на тему "Розробка та тестування даних моделей з Django/Flask: від визначення до PyTest валідації".

У межах роботи реалізовано дві незалежні ORM-системи (Django ORM та Flask-SQLAlchemy) для предметної області «Бібліотека», налаштовано автоматичні міграції, оптимізовано SQL-запити для усунення проблеми N+1, а також побудовано повноцінне тестове середовище за допомогою PyTest із використанням фікстур, параметризації, Mock-об'єктів та аналізу покриття коду (Coverage).

## Структура проекту

| Файл / Папка | Завдання | Опис |
| :--- | :---: | :--- |
| `django_app/library/models.py` | 1 | Реалізація Django ORM моделей (`Category`, `Author`, `Book`, `BookInventory`) із різними типами зв'язків та каскадним видаленням |
| `django_app/library/migrations/` | 2 | Файли автоматичних міграцій схеми бази даних для Django |
| `flask_app/models.py` | 3 | Декларативний опис моделей та проміжної таблиці Many-to-Many на Flask-SQLAlchemy |
| `flask_app/migrations/` | 3 | Скрипти міграцій бази даних Flask, згенеровані через Flask-Migrate (Alembic) |
| `django_app/library/views.py` | 4 | Ендпоінти для демонстрації оптимізації запитів (`select_related`, `prefetch_related`) та агрегації (`Count`) |
| `tests/conftest.py` | 5, 6 | Глобальні фікстури PyTest для ініціалізації тестових баз даних та ізольованих HTTP-клієнтів |
| `tests/test_django.py` | 5, 6, 7 | Тести моделей Django (унікальність, параметризація), HTTP-тести маршрутів та Mock-тестування (`@patch`) агрегації |
| `tests/test_flask.py` | 5, 6, 7 | Інтеграційні та ізольовані тести моделей, роутів та Mock-тести ORM-запитів для Flask-додатка |
| `pytest.ini` | - | Конфігураційний файл PyTest із налаштуваннями `pythonpath` та `DJANGO_SETTINGS_MODULE` |
